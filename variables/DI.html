    The DI namespace object contains static properties and methods for invoking Dynamic Icons element constructors and utility functions.

    To use any of these classes or functions, prefix them with DI (similar to how Math works). For example:

        let c = DI.round4p(Math.cos(angle));
        const textElement = new DI.StyledText({ text: "Hello World!" });

    The documentation here just provides links to the actual class/function references (also found in the left navigation bar). Click on the types (on right side) for full details.
